# Integrator Nuxt3

Nuxt3 проект каталога интеграций, созданный на основе макета из Figma.

## Описание

Это полнофункциональное веб-приложение для показа каталога интеграций:
- Современный дизайн по макету Figma
- Адаптивный отзывчивый дизайн
- Быстрая работа благодаря Nuxt3
- Простая навигация и расширяемость

## Файлы проекта

- `package.json` - зависимости проекта
- `nuxt.config.ts` - конфигурация Nuxt
- `app.vue` - корневой компонент
- `components/Header.vue` - шапка приложения
- `components/IntegrationCard.vue` - карточка интеграции
- `pages/index.vue` - главная страница

## Установка

```bash
# Клонирование
git clone https://github.com/Arrttyom/integrator-nuxt.git
cd integrator-nuxt

# Установка зависимостей
npm install
```

## Развитие

```bash
# Чинейный режим доразвития
npm run dev
```

Openhttp://localhost:3000

## Подготовка к production

```bash
# Сборка
npm run build

# Файлы будут в .output/public
```

## Публикация на GitHub Pages

```bash
# Генерация статического сайта
npm run generate

# Открывается .output/public в gh-pages
```
